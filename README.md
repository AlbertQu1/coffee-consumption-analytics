# ☕ Coffee Consumption Analytics

Data-driven coffee consumption intelligence platform built with Python, Pandas, and Machine Learning.

This project started as a simple personal coffee expense calculator and evolved into a modular analytics system capable of:

- consumption forecasting
- inventory tracking
- ROI analysis
- seasonality detection
- behavioral analytics
- machine learning predictions
- historical snapshot tracking

The goal of this project is to combine real-world personal data with analytical thinking and predictive modeling to simulate how business intelligence systems evolve over time.
